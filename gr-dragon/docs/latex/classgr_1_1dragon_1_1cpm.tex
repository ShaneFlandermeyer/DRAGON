Return the taps for an interpolating F\+IR filter (gr\+::filter\+::interp\+\_\+fir\+\_\+filter\+\_\+fff). 

These taps represent the phase response $g(k)$ for use in a C\+PM modulator, see also gr\+\_\+cpmmod\+\_\+bc.


\begin{DoxyParams}{Parameters}
{\em type} & The C\+PM type (Rectangular, Raised Cosine, Spectral Raised Cosine, Tamed FM or Gaussian). \\
\hline
{\em samples\+\_\+per\+\_\+sym} & Samples per symbol. \\
\hline
{\em L} & The length of the phase response in symbols. \\
\hline
{\em beta} & For Spectral Raised Cosine, this is the rolloff factor. For Gaussian phase responses, this the 3d\+B-\/time-\/bandwidth product. For all other cases, it is ignored.\\

Output\+: returns a vector of length {\itshape K} = {\ttfamily samples\+\_\+per\+\_\+sym} x {\ttfamily L}. This can be used directly in an interpolating F\+IR filter such as gr\+\_\+interp\+\_\+fir\+\_\+filter\+\_\+fff with interpolation factor {\ttfamily samples\+\_\+per\+\_\+sym}.

All phase responses are normalised s.\+t. $ \sum_{k=0}^{K-1} g(k) = 1$; this will cause a maximum phase change of $ h \cdot \pi$ between two symbols, where {\itshape h} is the modulation index.

The following phase responses can be generated\+:
\begin{DoxyItemize}
\item L\+R\+EC\+: Rectangular phase response.
\item L\+RC\+: Raised cosine phase response, looks like 1 -\/ cos(x).
\item L\+S\+RC\+: Spectral raised cosine. This requires a rolloff factor beta. The phase response is the Fourier transform of raised cosine function.
\item T\+FM\+: Tamed frequency modulation. This scheme minimizes phase change for rapidly varying input symbols.
\item G\+A\+U\+S\+S\+I\+AN\+: A Gaussian phase response. For a modulation index h = 1/2, this results in G\+M\+SK.
\end{DoxyItemize}

A short description of all these phase responses can be found in \mbox{[}1\mbox{]}.
