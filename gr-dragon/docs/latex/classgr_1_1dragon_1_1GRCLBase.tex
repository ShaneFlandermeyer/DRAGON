Initializes an Open\+CL instance. Specifically, the following steps are taken\+:
\begin{DoxyEnumerate}
\item Query the list of available platforms
\item Specify the device type (G\+PU, C\+PU, Accelerator, Any)
\item Choose whether to use the host pointer to access data (for C\+P\+Us) or copy the data to the Open\+CL device (G\+P\+Us, Accelerators, etc.)
\item If a specific device ID is specified, try to query that device. If not, choose the first platform that has the specified device type
\item Create a context from this device.
\item Create a command queue from the context. 
\end{DoxyEnumerate}
